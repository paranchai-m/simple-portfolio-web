# Portfolio Website Development Plan

## Objective
Create a modern, minimal portfolio website for Paranchai Malaikanok, a palynologist, emphasizing academic excellence and professional certifications.

## Scope
- Static HTML/CSS/JS website.
- Hybrid Neo-minimalist Bauhaus & Neo-Minimal theme.
- Sticky Top Navigation with integrated right-aligned "Contact Me" button.
- Publication section with justified block-text distribution in floating cards.
- Certifications section grouped by school with scroll-reveal animations.
